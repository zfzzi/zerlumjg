# 06 材料、铺装、构筑物与细部

## Purpose

让材料选择服务于尺度、使用、排水、耐久和场地气质，并在视觉表达中保持真实的构造关系。

## Read First

- 步行、车行、活动、亲水和服务区域的性能要求。
- 建筑材料、地方材料、既有构筑物和项目成本边界。
- 湿滑、热舒适、反光、耐污、冻融或盐雾等已知环境条件。
- 维护、更换、清洁和无障碍连续性。

## Design Actions

- 控制主材数量，建立主材、辅材和节点材料层级。
- 用铺装模数、收边、缝隙和坡向协调尺度与排水。
- 让座椅、栏杆、树池、台阶和构筑物共享一致的细部语言。
- 在触摸、坐靠、转角和高频磨损处优先检查安全与耐久。
- 通过真实接缝、厚度、基层关系和风化状态提升视觉可信度。

## Constraints

- 未提供产品、节点或工程资料时，不声称具体规格已满足项目要求。
- 不为追求效果图质感使用不适合场景的镜面、湿润或过度反射材质。
- 构筑物结构、防火、防腐和连接需专业复核。

## Failure Signals

- 材料种类过多且没有主次。
- 铺装图案独立于流线、尺度和排水。
- 家具像后期贴图，没有落位、基础或使用空间。

## Output Guidance

说明材料的使用部位、触感、尺度、耐久和维护原因。视觉提示词描述真实纹理、接缝、边缘、磨损和干湿状态。
