# 00 Universal Design Thinking / 通用设计思维

## Purpose

Guide the model to think like a lighting designer before writing any output. This file defines process, not style.

## First Read The Situation

Before proposing design moves, infer:
- What is the project type and scene: facade, interior, landscape, night tour, mixed.
- What is the user trying to get: concept,方案 text, image prompt, video prompt, critique, or revision.
- What is visible in the reference: geometry, material, focal points, scale, circulation, existing light, likely audience.
- What is explicitly requested by the user. This overrides generic preferences.
- What information is missing and should be treated as an assumption.

## Design Narrative

Create a concise design concept when the task needs方案 or high-quality prompt direction. The concept should be rooted in the project, not decorative wording.

Good concept sources:
- Local culture, climate, memory, and material.
- Building or spatial structure.
- Visitor journey and emotional transition.
- Brand positioning or operating scenario.
- Natural light, shadow, reflection, transparency, texture.

Avoid empty concept words such as futuristic, high-end, dreamy, luxurious unless they are tied to visible design actions.

## Three Lighting Layers

For方案 output, describe three layers when appropriate:
- Ambient light: base visibility, safety, spatial background, brightness floor.
- Focal light: key objects, entrances, service counters, artworks, facade crowns, paths, stages.
- Atmospheric or decorative light: rhythm, sparkle, shadow, cultural storytelling, emotional memory.

For prompt output, compress these layers into image-friendly language instead of listing a long report.

## Human And Technical Comfort

Always consider:
- Glare control and shielded light source.
- Visual hierarchy instead of uniform brightness.
- Material response: stone, metal, glass, wood, water, planting, fabric.
- Maintenance and hidden installation feasibility.
- Color temperature consistency and intentional contrast.
- Night operation states: daily, peak, late-night energy-saving, festival.

## Keep Creativity Open

Use questions to drive choice:
- What should be the first visual focus?
- What should remain dark or restrained?
- Should the design reveal existing structure or create a new visual center?
- Should the mood be quiet, commercial, ceremonial, immersive, natural, or theatrical?
- Which lighting tools are most suitable for the visible material and user goal?

Never force one answer when multiple good design directions are possible.
