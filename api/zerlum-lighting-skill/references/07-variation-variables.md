# 07 Variation Variables

## Purpose

Use the 差异化变量池 to prevent repetitive design. These are not fixed styles. They are design tensions that force the model to choose a project-specific direction.

不要把变量写成固定风格模板。Select one or two dominant variables based on the user image, project type, and intent, then let those variables shape lighting choices.

## Differentiation Variable Pool

- Structure reveal vs new visual focus.
- Quiet restraint vs theatrical drama.
- Material texture vs pure luminous atmosphere.
- Warm hospitality vs crisp modernity.
- Natural shadow vs artificial media expression.
- Hidden light source vs visible decorative fixture.
- Pedestrian intimacy vs urban landmark visibility.
- Static poetic scene vs dynamic operation sequence.
- Local cultural memory vs international minimal expression.
- Soft continuity vs sharp rhythm and contrast.
- Daily elegance vs festival spectacle.
- Human comfort vs iconic visual intensity.

## How To Use

For方案 output, name the chosen dominant tendency briefly and show how it affects ambient, focal, and atmospheric lighting.

For image or video prompts, do not output an analysis paragraph unless requested. Translate the chosen variable into visual language, for example:
- restrained warm indirect light, dark ceiling shadow, precise stone grazing.
- theatrical bamboo shadow projection, layered warm pools of light, hidden linear glow.
- quiet waterfront path light, low amber guide lights, dark planting background, subtle water reflection.

## Avoid

- Selecting variables randomly when the image already suggests a clear direction.
- Combining too many variables until the design loses focus.
- Reusing the same variable combination for all hotel, facade, landscape, or night-tour projects.
